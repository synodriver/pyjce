- 参考了 [pyjce](https://github.com/washingtown/PyJce)
- 原版只是实现了反序列化,此版本加入了序列化功能

- v0.1.0 大改 使用pydantic验证类型 自动读写jce结构体
- v0.1.1 弃用python3.7支持,只支持py3.8+
- v0.1.2 加入自动加密解密方法, 用于解析非递归的结构体, 复杂的仍然需要手动完成
  
    下一步: 完善单元测试
